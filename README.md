# After install projects

Requirements project's:

- python:latest
- nodejs:latest

## Install Back-end

- `cd api`
- `python -m venv venv`
- `venv\Scripts\activate`
- `pip install -r requirements.txt`

## Install Front-end

- `cd &{PROJECT_APP}`
- `npm install`

## Launch project

- `npm run start-api`
- `npm start`

### Clear DataBase

- `localhost:5000/messages/delete`
